# Travel Calendar

## Install

```
npm install
```

## Usage

```
mongod
```
```
mongoimport --db travel-calendar --collection travels --file ./resources/data/travel-calendar/travels.json
mongoimport --db travel-calendar --collection users --file ./resources/data/travel-calendar/users.json
```
```
node server
```
